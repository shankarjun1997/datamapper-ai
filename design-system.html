Export Controls

Available after reconciliation is complete and Gate 2 is approved. Three outputs: STM CSV, full XLSX report, and materialized BQ SQL script.

📄

Source-to-target mapping table in flat CSV format. Compatible with any ETL tool.

📊

XLSX Report

Full mapping report with confidence scores, business logic, and audit metadata.

⚙

Materialized BQ SQL

Ready-to-run CREATE OR REPLACE TABLE statement with properly sanitized BQ names.

Materialized SQL Code Block

The generated BQ SQL uses sanitized table/column names (snake_case, no reserved words, project.dataset.table format). Rendered in a dark code block with copy action.

generated_stm_session_1042.sql
-- Auto-generated by DataMapper Agentic Engine -- Session: 1042 · Generated: 2026-05-15T10:42:00Z -- Confidence avg: 76% · 75 mapped · 9 manual · 0 unmapped CREATE OR REPLACE TABLE `analytics-prod-442.warehouse_core.dim_customer` AS SELECT -- Direct mappings (high confidence) CAST(c.customer_id AS INT64) AS cust_id, UPPER(TRIM(c.first_name)) AS first_name, UPPER(TRIM(c.last_name)) AS last_name, DATE(c.signup_dt) AS created_date, -- Lookup mapping (medium confidence — reviewed) seg.segment_code AS segment_code, -- Audit columns (synthesized by L4) CURRENT_TIMESTAMP() AS _loaded_at, 'stm_session_1042' AS _source_run_id, ROW_NUMBER() OVER (PARTITION BY c.customer_id ORDER BY c.signup_dt DESC) AS _row_rank FROM `analytics-prod-442.source_raw.customers_v2` c LEFT JOIN `analytics-prod-442.warehouse_core.segment_lkp` seg ON c.acct_tier_cd = seg.src_code WHERE c._is_deleted IS FALSE;
